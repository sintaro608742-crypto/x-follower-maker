## 🐛 トラブルシューティングFAQ

### Q1: X連携ボタンをクリックしても何も起きない

**原因**: JavaScriptエラーまたはブラウザの問題

**解決方法**:
1. ブラウザのコンソールを開く（F12キー）
2. エラーメッセージを確認
3. ページをリロード（Ctrl+R / Cmd+R）
4. 別のブラウザで試す

### Q2: X連携後もダッシュボードに「X連携」ボタンが表示される

**原因**: セッション更新の遅延

**解決方法**:
1. ページをリロード（Ctrl+R / Cmd+R）
2. 一度ログアウトして再ログイン
3. それでも解決しない場合は連絡してください

### Q3: 投稿が `failed` ステータスになる

**原因**: Twitterトークンの有効期限切れまたはAPI制限

**解決方法**:
1. 投稿ページでエラーメッセージを確認
2. エラーが「User not found or Twitter not connected」の場合:
   - X連携を再実行
3. エラーが「Rate limit exceeded」の場合:
   - X API制限超過（1,500ツイート/月）
   - 投稿頻度を下げる

### Q4: フォロワーグラフが表示されない

**原因**: まだフォロワー記録が実行されていない

**解決方法**:
1. フォロワー記録は毎日8時に実行されます
2. 翌日8時以降にグラフが表示されます
3. すぐに確認したい場合は連絡してください

### Q5: 投稿内容を編集したい

**解決方法**:
1. `/posts` ページにアクセス
2. 編集したい投稿の「✏️ 編集」ボタンをクリック
3. 内容を編集して保存
4. 「✅ 承認」をクリックして承認

### Q6: 自動投稿を一時停止したい

**解決方法**:
1. `/posts` ページにアクセス
2. 投稿をすべて削除するか、承認を取り消す
3. 新しい投稿は生成されないため、自動投稿が停止します
